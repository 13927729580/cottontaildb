
# Header
-----------------------------------------------------------------------------
|   'HARE' (4 Bytes)   |    Version (2 Bytes)   |   Column Type (2 Bytes)   |
-----------------------------------------------------------------------------
|   ColumnSize (4 Bytes)            |    StripeSize (4 Bytes)               |
-----------------------------------------------------------------------------
|                              Flags (8 Bytes)                              |
-----------------------------------------------------------------------------
|                              Rows (8 Bytes)                               |
-----------------------------------------------------------------------------
|                              Deleted (8 Bytes)                            |
-----------------------------------------------------------------------------
|                              MaxRowId (8 Bytes)                           |
-----------------------------------------------------------------------------
|      Checksum (4 Bytes)            |       Offset 1st stripe (4 Bytes)    |
-----------------------------------------------------------------------------
|                                Statistics                                 |
|                                (Variable)                                 |
-----------------------------------------------------------------------------
|                                 Stripes                                   |
|                                (Variable)                                 |
-----------------------------------------------------------------------------

# Stripe
-----------------------------------------------------------------------------
|         'STRP' (4 Bytes)          |            ID (4 bytes)               |
-----------------------------------------------------------------------------
|         Rows (4 Bytes)            |         Checksum (4 bytes)            |
-----------------------------------------------------------------------------
|          Checksum (4 bytes)       |       Offset 1st data (4 bytes)       |
-----------------------------------------------------------------------------
|                           Statistics (8 Bytes)                            |
|                                                                           |
-----------------------------------------------------------------------------
|                              Rows (Variable)                              |
|                                                                           |
-----------------------------------------------------------------------------

# Row
-----------------------------------------------------------------------------
|                            Row ID (8 bytes)                               |
-----------------------------------------------------------------------------
|         Flags (4 Bytes)           |         Checksum (4 bytes)            |
-----------------------------------------------------------------------------
|                                 Data                                      |
|                              (Variable)                                   |
-----------------------------------------------------------------------------
